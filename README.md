# MTG Proxy Creator 🃏

Applicazione React per creare e stampare carte proxy MTG.

## Avvio rapido

```bash
npm install
npm run dev
```

## Funzionalità

- **Proxy Stampa**: carica immagini, drag & drop, riordina, genera PDF con segni di taglio
- **Token Creator**: editor token MTG con frame personalizzati, download PNG UHD

## Build produzione

```bash
npm run build
```
